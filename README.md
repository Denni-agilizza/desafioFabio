Projeto Inventário TI

---

Este projeto foi criado para atender os requisitos de um desafio técnico. O desafio consiste em criar um sistema básico de inventário onde o usuário digita as informações e o código armazena em um local storage.

---

Tecnologias utilizadas:
HTML
CSS
JavaScript
Node.js

---

Como Clonar e Rodar o Projeto

Este é o passo a passo para baixar e executar o sistema de Inventário de TI na sua máquina.

Pré-requisitos
Antes de começar, você precisa ter instalados no seu computador:
* [Git](https://git-scm.com/) (para clonar o repositório)
* [Node.js](https://nodejs.org/) (para rodar o servidor backend)

Passo a Passo

1. Clonar o repositório
Abra o terminal e rode o comando abaixo para baixar o código para a sua máquina:
```bash
git clone [https://github.com/Denni-agilizza/desafioFabio.git](https://github.com/Denni-agilizza/desafioFabio.git)

2. Entrar na pasta do projeto
Ainda no terminal, acesse a pasta que acabou de ser criada:

cd desafioFabio

3. Instalar as dependências
O projeto precisa de algumas bibliotecas (Express e CORS) para funcionar. Instale-as rodando:

npm install

4. Iniciar o servidor
Para que o sistema consiga salvar os dados no arquivo JSON, inicie o servidor local com o comando:

node server.js


5. Abrir o sistema
Com o servidor rodando, abra a pasta desafioFabio no explorador de arquivos do seu computador e dê um duplo clique no arquivo index.html. O sistema abrirá no seu navegador padrão e já estará pronto para uso.
